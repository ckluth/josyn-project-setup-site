Derived projections — the current intended TARGET picture (highest-value carry-forward)

Repo-world / taxonomy (docs/derived/repo-landscape.md): one concern per repo, poly-repo; four repo roles — artifact, concern, pilot-seat, platform-docs; repo name mirrors package namespace (JOSYN.* → kebab-case); dependencies form a strictly-tiered DAG of NuGet PackageReferences; independent SemVer per repo, bottom-up release ordering by tier; a pinned platform set = the exact version set that names "the platform right now".

Classification vocabulary (docs/derived/classification-vocabulary.md): slogan "family up front, axes in the back room, tiers are numbers"; four grouping nouns — concept, component, family, tier; primary classifier = family/sub-family ("sharpen, don't dissolve"); six axes — tier, stability, form, concern, reach, cardinality; tiers are computed numbers (0 primitives, 1 foundation, 2 libraries-and-contracts, 3 applications); retired words — helper, layer, loose "commons", loose "primitive", vague "category/type".
