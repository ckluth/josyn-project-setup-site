What it is

Authoritative architectural reference for JOSYN ("Job System Next") — a .NET platform executing scheduled jobs as isolated processes, orchestrated by a central scheduler over a named-pipe (JAP) protocol. No code — only architecture, ADRs, plans, sanity criteria, cross-cutting docs. Currently also the agentic pilot-seat (the repo agents read first) — explicitly flagged provisional. State: core repos/packaging settled; backend + scheduling mature; surface / JRP / Gateway stream still in active transition.

Folders: architecture/, solution-architecture/, decisions/ (45 ADRs), plans/ (13), proposals/ (1), guides/, sanity/, thinking/, repos/, README.md, ROADMAP.md, AGENTS.md, MAINTAINERS.md.
